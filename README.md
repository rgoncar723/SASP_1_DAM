# BioAcción - Unidos por la red, movidos por el verde

---

## Nuestro servicio:

Es una solución tecnológica que combina una aplicación web de gestión colaborativa con un sistema de riego inteligente y sensores de monitorización alimentados por energías renovables. Nuestro proyecto nace de la fusión colaborativa entre la tecnología IoT y la conservación medioambiental, con el objetivo de permitir un seguimiento en tiempo real del estado de la flora en entornos verdes.
